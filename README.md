# Meu Portfólio Profissional 🌐

Bem-vindo ao repositório do meu portfólio pessoal! 

Neste projeto, aplico as tecnologias que estou a aprender no curso de **Análise e Desenvolvimento de Sistemas do SENAI**.

## 🛠️ Tecnologias Utilizadas:
- **HTML5**: Estruturação semântica.
- **CSS3**: Design responsivo e moderno.
- **JavaScript**: Lógica de interação.

## 🎯 Objetivo:
Demonstrar a minha evolução técnica e facilitar o contacto para oportunidades de **Estágio** ou **Jovem Aprendiz** na área de TI.
